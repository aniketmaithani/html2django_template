*html2dt* links the static files present in the Django project *static*
folder to .html files.
Eg: `<img src="/img/abc.jpg">` converted to `<img src="{% static '/img/abc.jpg' %}">`

*Note*- Bash shell is needed for it's working.

####Installation####
`pip install html2dt`

####Usage####
`html2dt index.html`
